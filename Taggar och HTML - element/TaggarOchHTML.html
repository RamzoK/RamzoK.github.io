FRÅGOR OCH SVAR

  1. Vad används < br > till?
    2. < br > Sätter en line break. dvs att den lämnar utrymme mellan två textrader
  2. Vad används < !--...--> till i HTML?
    3. <--..--> kallas för Comment tag också. Det är för att man kan skriva kommentarer inom denna tag som inte syns på webbsidan, men existerar i koden. detta kan användas till anteckningar om dokumentet också.
  3. Varför skulle man vilja använda < a >-tagg?
    4. En < a >-tagg, också kallad för hyperlink tagg lägger till en länk i texten som skickar användaren från en webbsida till en annan.
  4. Vilken tagg skall du använda om du vill ha liten text?
    5. För att ha liten text skall man ange < small > - taggen
  5. Vilka taggar skall du använda om du vill ha en rubrik?
    6. För att ha en rubrik ska man använda < h > < /h >. Och om man vill ha mindre storlek kan man ange < h2 >, < h4 >, osv. Det blir bara mindre för var högre nummer.
  6. Vad kan du åstadkomma med en < i > - tag?
    7. < i > tag används för att framkalla italic fonten.
